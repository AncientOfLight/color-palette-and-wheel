# 🎨 Color Palette Generator & Interactive Color Wheel

A modern, professional web application featuring an advanced color palette generator and an interactive HTML5 Canvas-based color wheel component. Built with vanilla JavaScript, HTML5, and CSS3 (Tailwind CSS).

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Active-success?style=flat-square&logo=github)](https://ancientoflight.github.io/color-palette-and-wheel/)

[![JavaScript](https://img.shields.io/badge/JavaScript-ES6%2B-yellow?style=flat-square&logo=javascript)](/)

## 🌟 Features

### Color Palette Generator
- **Vibrant Random Palettes**: Generate 5-color palettes with a single click or spacebar press
- **Lock System**: Freeze individual colors to prevent changes during regeneration
- **Copy to Clipboard**: Instantly copy HEX codes with a single click
- **Export Functionality**: Download palettes as JSON for integration with design tools
- **Responsive Grid**: Works seamlessly on mobile and desktop browsers

### Interactive Color Wheel
- **360-Degree HSV Color Space**: Full spectrum selection with saturation and brightness control
- **Click & Drag Support**: Intuitive interaction with visual feedback
- **Touch Support**: Full mobile touch compatibility
- **Real-time Callbacks**: Get color updates during interaction
- **Multiple Output Formats**: HEX, RGB, HSL, and HSV color formats
- **Performance Optimized**: Smooth canvas rendering with no lag

### Advanced Features
- **Intelligent Search Bar**: Filter palettes by name, color code, or emotion/mood keywords
- **12 Color Categories**: Warm, Cold, Vintage, Retro, Gold, Nature, Space, Cyberpunk, Ocean, Forest, Sunset, and more
- **Sentiment-based Filtering**: AI-like emotion matching (e.g., "dreamy", "mystical", "tropical")
- **Dark Mode UI**: Futuristic midnight aesthetic with glassmorphism design
- **Live Statistics**: Track generated and saved color palettes
- **Local Storage**: Persist palette data across sessions

## 🎡 Color Wheel Component Usage

The Color Wheel is a reusable, modular component that can be integrated into any project.

### Basic Implementation

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Color Wheel Demo</title>
    <link rel="stylesheet" href="color-wheel.css">
</head>
<body>
    <div id="color-wheel"></div>
    
    <script src="color-wheel.js"></script>
    <script>
        const wheel = new ColorWheel({
            container: '#color-wheel',
            size: 300,
            initialColor: '#FF0000'
        });
    </script>
</body>
</html>
```

### Advanced Configuration

```javascript
const wheel = new ColorWheel({
    // DOM container selector
    container: '#color-wheel',
    
    // Wheel size in pixels (default: 300)
    size: 300,
    
    // Initial HEX color (default: '#FF0000')
    initialColor: '#3B82F6',
    
    // Callback fired during color selection (dragging)
    onColorChange: (color) => {
        console.log('Current HEX:', color.hex);
        console.log('Current RGB:', color.rgb);
        console.log('Current HSL:', color.hsl);
        console.log('Current HSV:', color.hsv);
    },
    
    // Callback fired when user releases mouse/touch
    onColorSelect: (color) => {
        console.log('Selected color:', color.hex);
        // Save to database, update UI, etc.
    },
    
    // Show saturation reference rings (default: true)
    showSaturationRadius: true,
    
    // Enable touch support on mobile (default: true)
    enableTouch: true
});
```

### API Reference

#### Methods

```javascript
// Set color programmatically
wheel.setColor('#FF5733');

// Get current color in all formats
const color = wheel.getColor();
// Returns: {
//   hex: '#FF5733',
//   rgb: { r: 255, g: 87, b: 51 },
//   hsl: { h: 10, s: 100, l: 60 },
//   hsv: { h: 10, s: 80, v: 100 }
// }

// Clean up and remove component
wheel.destroy();
```

#### Color Formats

The Color Wheel returns colors in multiple formats for flexibility:

- **HEX**: `#FF0000` (most common for web)
- **RGB**: `{ r: 255, g: 0, b: 0 }` (CSS-compatible)
- **HSL**: `{ h: 0, s: 100, l: 50 }` (for CSS `hsl()` function)
- **HSV**: `{ h: 0, s: 100, v: 100 }` (HSV color space)

## 🌐 GitHub Pages Deployment

The project is configured to automatically deploy to GitHub Pages when you push to the `main` branch.

### Setup Instructions

1. **Vite Configuration** is already set with the correct base path:
   ```javascript
   // vite.config.js
   export default defineConfig({
     plugins: [react()],
     base: '/color-palette-and-wheel/',
   })
   ```

2. **GitHub Actions Workflow** - Create `.github/workflows/deploy.yml`:

```yaml
name: Build and Deploy to GitHub Pages

on:
  push:
    branches:
      - main
  pull_request:
    branches:
      - main

permissions:
  contents: read
  pages: write
  id-token: write

concurrency:
  group: 'pages'
  cancel-in-progress: true

jobs:
  build:
    name: Build Application
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - uses: actions/setup-node@v4
        with:
          node-version: '18'
          cache: 'npm'
      
      - run: npm ci
      - run: npm run lint || true
      - run: npm run build
      
      - uses: actions/upload-pages-artifact@v3
        with:
          path: 'dist'

  deploy:
    name: Deploy to GitHub Pages
    needs: build
    runs-on: ubuntu-latest
    if: github.ref == 'refs/heads/main' && github.event_name == 'push'
    
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    
    steps:
      - uses: actions/deploy-pages@v4
        id: deployment
```

3. **Configure GitHub Pages Settings**:
   - Go to Settings → Pages
   - Source: Deploy from a branch
   - Branch: `gh-pages` (auto-created by workflow)
   - Folder: `/ (root)`

4. **Push to Main Branch**:
   ```bash
   git add .
   git commit -m "Setup GitHub Pages deployment"
   git push origin main
   ```

5. **Monitor Deployment**:
   - Check Actions tab in your repository
   - Wait for workflow to complete
   - Visit `https://ancientoflight.github.io/color-palette-and-wheel/`

```javascript
const wheel = new ColorWheel({
    container: '#brand-color-wheel',
    size: 250,
    onColorSelect: (color) => {
        // Update CSS variables
        document.documentElement.style.setProperty(
            '--brand-primary',
            color.hex
        );
        
        // Save to database
        saveBrandColor(color.hex);
    }
});
```

### Example 2: Palette Generator Integration

```javascript
const colorWheels = [];

document.querySelectorAll('[data-color-wheel]').forEach((el, i) => {
    const wheel = new ColorWheel({
        container: el,
        size: 200,
        onColorSelect: (color) => {
            savePaletteColor(i, color.hex);
        }
    });
    colorWheels.push(wheel);
});
```

### Example 3: Live Preview

```javascript
const wheel = new ColorWheel({
    container: '#color-wheel',
    onColorChange: (color) => {
        // Update preview in real-time
        document.getElementById('preview').style.background = color.hex;
        document.getElementById('hex-display').textContent = color.hex;
        document.getElementById('rgb-display').textContent = 
            `rgb(${color.rgb.r}, ${color.rgb.g}, ${color.rgb.b})`;
    }
});
```
